urlScraper.js

This is a function written to scrape the ingredient instructions off a website (a limitation of Edamam's API, it just returns the URL.) This is due to CORS, a mechanism that allows restricted resources on a web page to be accessed from another domain outside of it. Due to many of these webpages not allowing for their information to be scraped, they don't have CORS headers in their HTTP headers, preventing any unauthorized access to their information. Further work is to investigate how to properly webscrape and get instructions off of the site to make this website fully functional :).

The application does work for local testing however, at the moment it returns the instructions set (as that was the functionality needed.) The .js works by using Cheerio, a node.js library for manipulating HTML, and Puppeteer, a node.js library that provides a high-level API to control a chrome browser, simulating a HTTP request and avoiding that CORS mechanism.

To run the JS file, all you need to do is run node urlScraper.js, put in the URL you want to scrape, and it should work.
